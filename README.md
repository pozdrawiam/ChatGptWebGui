# ChatGptWebGui
Web frontend for OpenAi ChatGPT.
Chat conversations by own api key.
Blazor WebAssembly.

## Dependencies

- .Net 8

## Run from source
```sh
dotnet run --project Cwg.WebGui
```

## Publish
```sh
dotnet publish Cwg.WebGui -c Release
```
